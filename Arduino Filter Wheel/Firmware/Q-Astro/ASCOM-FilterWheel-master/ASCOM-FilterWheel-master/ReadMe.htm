ASCOM QuidneArduino FilterWheel Driver (C#)



You have just installed the QuidneArduino Filterwheel Driver

This driver allows you to use ASCOM compatible software to control an Arduino powered filterwheel. Presumably you have already installed the motor control software onto the Arduino. The source code for Arduino is available here <> as is the ASCOM C# driver code. The code was developed for my specific setup and needs to tweaked for your DIY project. This is not an official ASCOM endorsed driver.



ASCOM Initiative



The ASCOM Initiative consists of a group of astronomy software developers and instrument vendors whose goals are to promote the driver/client model and scripting automation.
